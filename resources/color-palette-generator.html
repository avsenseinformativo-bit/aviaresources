Color Theory Basics

Understanding basic color theory helps you create better palettes:

The Color Wheel

The color wheel organizes colors in a circle, showing relationships between primary, secondary, and tertiary colors:

  • Primary Colors: Red, Blue, Yellow - cannot be created by mixing
  • Secondary Colors: Green, Orange, Purple - created by mixing primaries
  • Tertiary Colors: Red-Orange, Yellow-Green, etc. - mixing primary and secondary

Color Harmony Rules

60% Dominant Color
30% Secondary Color
10% Accent Color

This 60-30-10 rule creates balanced, professional-looking designs.

Popular Color Scheme Types

Monochromatic

Uses different shades, tints, and tones of a single color. Creates a cohesive, elegant look.

Example: Navy blue (#1e3a8a), Medium blue (#3b82f6), Light blue (#93c5fd), Pale blue (#dbeafe)

Best for: Minimalist designs, professional brands

Complementary

Uses colors opposite each other on the color wheel. Creates high contrast and visual interest.

Example: Blue (#3b82f6) and Orange (#fb923c)

Best for: Call-to-action buttons, eye-catching designs

Analogous

Uses colors next to each other on the color wheel. Creates harmonious, natural-looking palettes.

Example: Blue (#3b82f6), Blue-Green (#06b6d4), Green (#10b981)

Best for: Nature themes, calming designs

Triadic

Uses three colors evenly spaced on the color wheel. Creates vibrant, balanced palettes.

Example: Red (#ef4444), Yellow (#eab308), Blue (#3b82f6)

Best for: Playful brands, children's products

Color Accessibility Guidelines

Creating accessible color palettes ensures everyone can use your design:

Contrast Ratios (WCAG Standards)

4.5:1 Normal Text Minimum
3:1 Large Text Minimum
7:1 AAA Level (Enhanced)

Best Practices for Accessibility

  • Don't rely on color alone: Use icons, labels, or patterns too
  • Test with color blindness simulators: Ensure 8% of men can use your design
  • Provide sufficient contrast: Especially for text and interactive elements
  • Use color consistently: Same colors should mean the same things
AI Prompt for Accessible Palettes:
"Create a WCAG AAA compliant color palette for a [type] website. Ensure all text colors have at least 7:1 contrast ratio with backgrounds. Include hex codes and contrast ratios."

Tools for Testing Accessibility

  • WebAIM Contrast Checker: Test color contrast ratios
  • Colorblind Web Page Filter: Simulate different types of color blindness
  • WAVE: Comprehensive accessibility evaluation
